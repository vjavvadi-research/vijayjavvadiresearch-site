About

Vijay P. Javvadi is an independent researcher focusing on artificial intelligence driven software testing, intelligent automation systems, and predictive quality engineering. His work explores how machine learning, repository analytics, and autonomous testing systems can transform traditional software quality assurance into intelligent quality engineering platforms.

His research focuses on developing AI-driven frameworks that predict software defects, generate automated test scenarios, and improve software reliability through intelligent testing strategies.

Research Areas

Research Vision

The objective of this research program is to advance the field of AI-Driven Intelligent Quality Engineering. Traditional software testing methods rely heavily on manual test design and static automation frameworks. This research explores how artificial intelligence, machine learning, and repository analytics can transform testing into predictive and autonomous quality engineering systems.

The proposed research focuses on predicting software defects, generating intelligent test cases, optimizing test execution, and continuously improving testing strategies using data-driven insights.

  • AI-Driven Test Automation
  • Machine Learning for Defect Prediction
  • Autonomous Test Generation
  • AI-Powered Quality Analytics
  • Security Testing Automation
  • Data Testing Automation
  • Mobile Test Automation
