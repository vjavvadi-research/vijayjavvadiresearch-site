Vijay Prasad Javvadi

Independent Researcher in AI-Driven Software Quality Engineering

My research focuses on applying artificial intelligence and machine learning to improve software reliability, testing efficiency, and quality engineering practices in large-scale software systems. Through the integration of repository analytics, predictive modeling, and intelligent automation frameworks, this research program aims to develop data-driven approaches for defect prediction and autonomous testing.

Research Focus Areas

AI-Driven Test Automation

Developing intelligent testing frameworks capable of adaptive execution, self-healing automation, and automated test generation using machine learning.

Software Defect Prediction

Applying machine learning models to software repository metrics to identify defect-prone modules and improve risk-based testing strategies.

Quality Analytics

Leveraging development data, commit history, and code metrics to build predictive analytics systems for software quality.

Autonomous Testing Systems

Exploring next-generation testing platforms capable of automatically generating, executing, and optimizing test scenarios.

Research Program

The objective of this research initiative is to transform traditional software testing practices into intelligent, predictive systems powered by artificial intelligence and machine learning. This research program investigates how repository analytics, predictive defect detection, and autonomous testing frameworks can significantly improve software reliability and reduce production failures.
