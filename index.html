About

Vijay P. Javvadi is an independent researcher focusing on artificial intelligence driven software testing, intelligent automation systems, and predictive quality engineering. His work explores how machine learning, repository analytics, and autonomous testing systems can transform traditional software quality assurance into intelligent quality engineering platforms.

His research focuses on developing AI-driven frameworks that predict software defects, generate automated test scenarios, and improve software reliability through intelligent testing strategies.

  • Machine Learning for Defect Prediction

  • Security Testing Automation
