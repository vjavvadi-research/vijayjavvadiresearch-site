AI-Driven Quality Engineering Research Program

This research program investigates how artificial intelligence can improve software testing, software reliability, and modern quality engineering practices.

The program combines repository mining, machine learning, automated testing, and software analytics to build predictive and autonomous quality engineering systems.

Research Themes

  • AI-Driven Software Defect Prediction Using Repository Analytics
  • AI-Driven Self-Healing Test Automation
  • Autonomous Regression Testing Systems
  • LLM-Based Test Case Generation
  • Intelligent API Contract Testing
  • Cloud-Based Distributed Testing Platforms
  • AI-Powered Quality Analytics for Software Systems
