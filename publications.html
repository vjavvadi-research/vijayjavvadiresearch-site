Publications

AI-Driven Software Quality Engineering Research
This research portfolio focuses on artificial intelligence applications in software testing, quality engineering, and intelligent automation systems. The work explores machine learning, large language models, and intelligent analytics to improve reliability and efficiency in large-scale software systems.
AI-Driven Software Defect Prediction Using Repository Analytics ▶

This research proposes a machine learning framework that analyzes repository metrics including commit activity, code churn, and file modification history to identify defect-prone software modules. The approach enables predictive risk analysis for large software projects and supports intelligent test prioritization strategies.

AI-Driven Self-Healing Test Automation Framework ▶

This research introduces a self-healing test automation architecture that automatically adapts to user interface changes during test execution. Using DOM similarity detection and machine learning models, the system dynamically identifies alternative element locators to prevent automation failures caused by UI updates.

Autonomous Regression Testing Systems ▶

This work explores autonomous regression testing systems capable of automatically prioritizing test suites based on historical defect patterns and repository analytics. The system dynamically selects high-risk tests to improve testing efficiency while maintaining strong defect detection.

LLM-Based Test Case Generation from Requirements ▶

This research investigates the use of large language models to automatically generate software test cases from system requirements, user stories, and functional specifications. The framework translates textual requirements into structured automated test scenarios.

Intelligent API Contract Testing for Microservices ▶

This study proposes an intelligent contract testing framework designed for microservice architectures. The system analyzes API specifications and detects compatibility issues between services to prevent integration failures in distributed systems.

AI-Powered Quality Analytics for Software Systems ▶

This research explores the use of artificial intelligence to analyze software quality metrics across development pipelines. The framework identifies patterns in test results, defect data, and development activity to generate predictive quality insights.

Machine Learning for Intelligent Test Prioritization ▶

This research proposes a machine learning approach to optimize test execution order in continuous integration environments. By analyzing historical test failures and code changes, the system prioritizes tests that are most likely to detect defects.

Autonomous UI Testing Using Computer Vision ▶

This work investigates computer vision techniques for autonomous UI testing. Instead of relying on traditional element locators, the system uses visual recognition models to identify UI components based on screen layout and appearance.

AI-Driven Continuous Testing Framework ▶

This research presents a continuous testing architecture that integrates artificial intelligence into CI/CD pipelines. The system dynamically selects test strategies based on risk prediction models and development activity patterns.

Cloud-Based Distributed Testing Platforms ▶

This research explores scalable cloud architectures for large-scale distributed testing. The platform enables parallel execution of test suites across cloud infrastructure to accelerate quality assurance processes.

AI-Assisted Root Cause Analysis for Software Failures ▶

This research investigates machine learning models capable of automatically identifying the root causes of software failures. The framework analyzes logs, error traces, and test failures to assist engineers in diagnosing defects faster.

Intelligent Test Data Generation Using AI ▶

This study proposes a framework for automatically generating test data using artificial intelligence models. The system produces diverse input datasets that improve test coverage and expose hidden defects.

Predictive Quality Engineering for Large Software Systems ▶

This research introduces predictive quality engineering models that analyze historical development data to forecast future quality risks in large software systems.

AI-Driven Automation Framework for Enterprise Applications ▶

This work proposes a modular automation framework that integrates artificial intelligence techniques to improve test maintenance and scalability for enterprise software applications.

Intelligent Software Testing Ecosystems ▶

This research explores the concept of intelligent testing ecosystems where artificial intelligence systems continuously analyze development activity and automatically adapt testing strategies.
