AI-Driven Software Defect Prediction Using Repository Analytics and Machine Learning

This research proposes a machine learning framework for predicting defect-prone software modules using repository-derived development metrics such as commit frequency, code churn, and developer participation.

The study evaluates Logistic Regression, Decision Tree, and Random Forest classifiers and demonstrates that machine learning models can support risk-based testing strategies in large-scale software systems.

AI-Driven Self-Healing Test Automation

This research investigates how intelligent automation systems can adapt test execution flows, recover from UI changes, and improve software testing resilience.

Autonomous Regression Testing Systems

Research focused on autonomous testing frameworks capable of selecting, generating, and prioritizing regression tests using predictive quality signals.
