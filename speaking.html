AI-Driven Software Defect Prediction Using Repository Analytics
Research Talk · Software Engineering Community

This talk presents research on predicting defect-prone software modules using machine learning models and repository mining. The work demonstrates how development metrics such as code churn, commit frequency, and file age can help identify risk areas in large software systems.

Autonomous Test Case Generation Using Large Language Models
Technical Workshop · AI in Software Testing

This presentation explores the use of large language models for automated test case generation. The talk highlights how AI can assist testers in generating meaningful test scenarios from software requirements and specifications.
