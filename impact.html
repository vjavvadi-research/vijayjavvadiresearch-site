Research Impact

Advancing Intelligent Software Quality Engineering
The research contributions presented on this platform focus on advancing AI-driven automation, predictive analytics for software quality, and intelligent testing systems. These innovations aim to improve reliability, efficiency, and scalability in modern software engineering practices.
AI-Driven Defect Prediction for Software Systems

Research on machine learning-based defect prediction demonstrates how repository analytics can be used to identify defect-prone modules before software failures occur. This work contributes to improving software reliability and supports risk-based testing strategies.

Autonomous Test Case Generation Using Large Language Models

This research explores the use of large language models to automatically generate test cases from software specifications. The work helps reduce manual test design effort and enables scalable testing for complex software systems.

AI-Powered Continuous Testing Platforms

This project investigates the integration of machine learning models within continuous integration pipelines to support predictive quality analytics and automated testing prioritization.

Self-Healing Test Automation Frameworks

Self-healing test automation systems dynamically adjust test scripts when UI or API changes occur. This research helps reduce test maintenance costs and increases stability of automated testing pipelines.

Intelligent Risk-Based Testing Using Machine Learning

This research applies machine learning techniques to identify high-risk areas in software systems and prioritize testing activities. The approach helps teams focus testing efforts on components most likely to fail.
