Advancing Intelligent Software Quality Engineering
The research contributions presented on this platform focus on advancing AI-driven automation, predictive analytics for software quality, and intelligent testing systems. These innovations aim to improve reliability, efficiency, and scalability in modern software engineering practices.

Risk-based testing using ML defect prediction identifies the top 10% highest-risk files that contain 43.2% of all defects. Deployed as a production FastAPI microservice integrated with GitHub Actions CI/CD pipelines.

68% Reduction in Test Design Effort via LLM Generation

The RAITG framework converts natural-language requirements into executable test cases with rule-based verification. Manual baseline of 184 hours reduced to 58.9 hours. Requirement coverage improved from 71.2% to 94.1% with a 96.3% first-pass verification rate.
