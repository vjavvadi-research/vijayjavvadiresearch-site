Vijay P. Javvadi – AI-Driven Test Automation and Intelligent Quality Engineering

Research Contributions

Vijay P. Javvadi’s research focuses on developing intelligent testing systems that apply artificial intelligence, machine learning, and repository analytics to improve software quality engineering. His work explores predictive testing systems capable of identifying software defects, generating autonomous test scenarios, and optimizing test execution strategies.

Key Research Contributions

  • Development of AI-driven defect prediction models that analyze software repository data to identify high-risk software modules.
  • Design of autonomous test generation frameworks capable of creating intelligent test scenarios based on predicted software risks.
  • Architecture of intelligent quality engineering platforms that integrate machine learning analytics with continuous testing pipelines.
  • Research on machine learning approaches for predicting defect-prone software features and developer risk patterns.
  • Development of AI-powered automation strategies for UI, API, mobile, data, and security testing.

Research Impact

These contributions aim to transform traditional software testing methods into predictive and autonomous quality engineering systems. By integrating artificial intelligence into software testing workflows, organizations can improve defect detection, optimize testing resources, and enhance software reliability in large-scale enterprise systems.

Research Areas

  • AI-Driven Test Automation
  • Machine Learning for Defect Prediction
  • Autonomous Test Generation
  • AI-Powered Quality Analytics
  • Security Testing Automation
  • Data Testing Automation
  • Mobile Test Automation
