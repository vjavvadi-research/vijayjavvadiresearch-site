AI Systems and Software Quality Engineering Platforms
This page presents research implementations and experimental platforms developed as part of the AI-Driven Software Quality Engineering research program.
TestForge AI — Intelligent Test Automation Platform ▶

An enterprise-grade platform that generates production-ready test automation frameworks, prioritizes tests using defect prediction, and manages distributed test execution with intelligent risk scoring. Combines framework generation, LLM-powered test creation, and ML-driven prioritization in a unified platform.

Technology Stack:
Next.js 14, React 18, TypeScript, Express.js, FastAPI, SQLite, Redis, Docker Compose, Anthropic Claude API, scikit-learn, XGBoost

Status: In Active Development

AI Defect Prediction Platform ▶

Machine learning system that predicts defect-prone software modules using repository analytics. Evaluated 6 classifiers on 284,676 instances across 5 open-source repositories. Random Forest achieves AUC 0.904 with 4.3x improvement in defect detection efficiency via risk-based testing.

Technology Stack:
Python, scikit-learn, XGBoost, GitPython, pandas, SMOTE, FastAPI

Status: Completed — Production Deployed

Test Case Generation Service (RAITG) ▶

FastAPI microservice that converts natural-language requirements into executable test cases using Claude LLM with prompt engineering and deterministic rule-based verification. Supports pytest, Gherkin, Playwright, and Postman output formats. Achieves 94.1% requirement coverage with 96.3% first-pass verification rate.

Technology Stack:
Python, FastAPI, Anthropic Claude API, PyYAML, Pydantic, Docker

Status: Completed — Production Deployed

Research Paper
Test Prioritization Service ▶

Production REST service that scores files by defect probability using a Gradient Boosting model trained on repository metrics. Integrates with CI/CD pipelines to reorder test execution by risk rank. Response latency under 30ms per request.

Technology Stack:
Python, FastAPI, scikit-learn, joblib, GitPython, structlog, Docker

Status: Completed — Production Deployed

Self-Healing Test Automation Framework ▶

Intelligent automation framework that recovers broken Selenium locators at runtime using DOM similarity analysis and ML-assisted ranking. Uses 8 feature families (tag, ID, name, class, text, placeholder, ARIA, path) with a hybrid heuristic + logistic regression ranking approach.

Technology Stack:
Java, TypeScript, Selenium WebDriver, Playwright, DOM Analysis, Logistic Regression

Status: In Development

AI Governance Framework ▶

Unified data governance layer for all AI quality services. Provides payload size controls (512KB default), content redaction via regex patterns, author privacy hashing, prompt versioning and allowlists, and service-specific configuration for git analysis and test generation.

Technology Stack:
Python, PyYAML, pytest

Status: Completed

Production Test Automation Reference Framework ▶

Full-featured UI and API test automation framework with BDD support, Page Object Model pattern, parallel execution, and multi-environment configuration. Includes database connectors (PostgreSQL, Snowflake, Oracle) and cloud integration (AWS S3/Secrets Manager, Azure Blob/Key Vault).

Technology Stack:
TypeScript, Playwright, Cucumber-JS, Axios, Allure Reporting, GitHub Actions, Jenkins

Status: Completed
