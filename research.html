Research Areas

Research domains in AI-Driven Software Quality Engineering
This research program explores how artificial intelligence, machine learning, and software repository analytics can transform modern software testing and quality engineering practices. The research focuses on predictive defect detection, intelligent automation systems, and autonomous testing frameworks designed to improve software reliability in large-scale software systems.

Repository Analytics & Defect Prediction

Research on machine learning models that analyze software repository metrics such as commit history, developer activity, and code evolution to identify defect-prone software modules.

Intelligent Test Automation

Development of AI-driven automation frameworks capable of self-healing test scripts and adaptive test execution in rapidly evolving software applications.

Autonomous Testing Systems

Exploration of autonomous testing platforms that automatically generate, prioritize, and execute regression tests using repository analytics and predictive models.

AI-Driven Test Generation & API Reliability

Research on intelligent systems that use large language models and machine learning techniques to generate test cases and validate API contracts in distributed software systems.
