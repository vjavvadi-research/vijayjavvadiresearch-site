Research Areas

Key research domains in AI-driven software quality engineering
This research program investigates how artificial intelligence, machine learning, and repository analytics can transform traditional software quality engineering practices into intelligent, predictive, and autonomous testing systems. The research focuses on developing data-driven approaches that improve software reliability, automate testing processes, and enable intelligent decision making in modern software development environments.

AI-Driven Test Automation

Research on intelligent automation frameworks capable of adaptive test execution, self-healing test scripts, and automated test generation using machine learning techniques.

Machine Learning for Defect Prediction

Applying machine learning models to software repository analytics to identify defect-prone modules and improve risk-based testing strategies in large-scale software systems.

Autonomous Testing Systems

Exploring next-generation testing platforms capable of automatically generating, prioritizing, and executing test cases based on historical development data and system behavior.

Intelligent Quality Analytics

Developing predictive analytics techniques that analyze software development metrics, commit history, and testing data to improve software reliability and quality assurance processes.
