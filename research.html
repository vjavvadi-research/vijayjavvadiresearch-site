Research Areas

Research domains in AI-Driven Software Quality Engineering
This research program explores how artificial intelligence, machine learning, and software repository analytics can transform modern software testing and quality engineering practices. The research focuses on predictive defect detection, intelligent automation systems, and autonomous testing frameworks designed to improve software reliability in large-scale software systems.

📊 Repository Analytics & Defect Prediction

Using repository mining and machine learning models to identify defect-prone software modules and support risk-based testing strategies in large-scale software systems.

🔧 AI-Driven Self-Healing Test Automation

Developing intelligent automation frameworks capable of detecting and repairing failing automated test scripts caused by UI changes using machine learning techniques.

🤖 Autonomous Regression Testing Systems

Designing testing platforms that automatically generate, prioritize, and execute regression tests based on repository analytics and defect prediction models.

🧠 LLM-Based Test Case Generation

Applying large language models to automatically generate software test cases from requirements, user stories, and application specifications.

🔗 Intelligent API Contract Testing

Developing AI-driven techniques to validate API contracts and detect compatibility issues between microservices in distributed software systems.
