Research Focus

The research program investigates how artificial intelligence, machine learning, repository analytics, and intelligent automation can transform software testing into predictive quality engineering.

This work focuses on using software development history, machine learning models, and autonomous execution systems to identify defect-prone software modules, generate targeted test strategies, and support data-driven software reliability improvement.

Core Research Domains

Software Defect Prediction

Predicting defect-prone modules using repository metrics, machine learning models, and software evolution patterns.

Intelligent Test Automation

Building AI-assisted testing systems capable of adapting to application behavior and optimizing execution strategies.

Autonomous Testing

Research on self-directed testing frameworks that can generate, select, and execute high-value tests with minimal manual effort.
